## 📋 ภาพรวมโครงการ

โปรเจกต์นี้เป็นระบบจัดการงานวิ่งมาราธอนที่ช่วยให้ผู้จัดงานสามารถจัดการข้อมูลของนักวิ่งที่ลงทะเบียน ติดตามสถานะการชำระเงิน และจัดการข้อมูลต่างๆ อย่างมีประสิทธิภาพ

## 🔧 โครงสร้างโปรเจกต์

```
marathon_project/
├── 📁 admin/                 # หน้าสำหรับผู้ดูแลระบบ
│   ├── admin_dashboard.php   # แดชบอร์ดผู้ดูแล
│   ├── admin_login.php       # หน้าเข้าสู่ระบบผู้ดูแล
│   ├── dashboard.php         # หน้าควบคุม
│   ├── edit_reg.php          # แก้ไขข้อมูลการสมัคร
│   ├── delete_process.php    # ลบข้อมูล
│   └── update_process.php    # อัปเดตข้อมูล
├── 📁 assets/                # ไฟล์สินทรัพย์
├── 📁 config/                # การตั้งค่าระบบ
│   └── db.php               # การเชื่อมต่อฐานข้อมูล
├── 📄 index.php              # หน้าหลัก
├── 📄 login_form.php         # หน้าเข้าสู่ระบบผู้ใช้
├── 📄 login_process.php      # ประมวลผลการเข้าสู่ระบบ
├── 📄 logout.php             # ออกจากระบบ
├── 📄 profile.php            # หน้าโปรไฟล์ผู้ใช้
├── 📄 register.php           # หน้าลงทะเบียน
├── 📄 process_register.php   # ประมวลผลการลงทะเบียน
├── 📄 summary.php            # สรุปการลงทะเบียน
├── 📄 test_conn.php          # ทดสอบการเชื่อมต่อ
├── 📄 running.sql            # ไฟล์ฐานข้อมูล
└── 📄 README.md              # คู่มือการใช้งานนี้
```

## ⚙️ การติดตั้งระบบ

### ข้อกำหนดเบื้องต้น
- 🌐 Web Server (Apache/Nginx)
- 🐘 PHP 7.4 ขึ้นไป
- 🗄️ MySQL/MariaDB
- 🌐 Web Browser

### ขั้นตอนการติดตั้ง

1. **ดาวน์โหลดโปรเจกต์** 📥
   ```bash
   git clone https://github.com/your-repo/marathon_project.git
   ```

2. **ติดตั้งฐานข้อมูล** 🗄️
   - สร้างฐานข้อมูลใหม่ชื่อ `running`
   - นำเข้าไฟล์ `running.sql` ไปยังฐานข้อมูล

3. **ตั้งค่าการเชื่อมต่อฐานข้อมูล** 🔌
   - แก้ไขไฟล์ `config/db.php` ตามการตั้งค่าของคุณ:
   ```php
   $host = "localhost";
   $user = "root";
   $pass = "your_password";
   $dbname = "running";
   ```

4. **เปิดใช้งานเว็บไซต์** 🚀
   - วางไฟล์ทั้งหมดในไดเรกทอรี web server
   - เปิดเบราว์เซอร์แล้วไปที่ `http://localhost/marathon_project`

## 🔐 การใช้งานระบบ

### ผู้ใช้งานทั่วไป

1. **ลงทะเบียน** 📝
   - ไปที่หน้า `register.php`
   - กรอกข้อมูลส่วนตัว
   - เลือกประเภทการแข่งขัน
   - ยืนยันการลงทะเบียน

2. **เข้าสู่ระบบ** 👤
   - ไปที่หน้า `login_form.php`
   - ใช้เลขบัตรประชาชนเป็นชื่อผู้ใช้
   - ใช้รหัสผ่านที่ตั้งไว้ตอนลงทะเบียน

3. **ดูข้อมูลส่วนตัว** 👤
   - หลังเข้าสู่ระบบ ไปที่ `profile.php`
   - ดูข้อมูล BIB และสถานะการชำระเงิน

### ผู้ดูแลระบบ

1. **เข้าสู่ระบบผู้ดูแล** 👨‍💼
   - ไปที่หน้า `admin/admin_login.php`
   - ใช้บัญชีผู้ดูแลระบบ
   username: admin01
   password: admin123

2. **จัดการข้อมูล** 📊
   - ดูรายชื่อผู้ลงทะเบียนทั้งหมด
   - แก้ไขสถานะการชำระเงิน
   - จัดการข้อมูลการสมัคร
